# 心理健康助手 Mental Health Assistant

一个基于 **vue3 + AI** 开发的心理健康辅助项目，
为用户提供轻量、温暖、便捷的心理支持服务。

## 项目功能
- 🌤 AI 智能情绪疏导与陪伴对话
- 📝 心情日记 / 情绪记录
- 🧪 心理状态自评量表/小测试
- 🎧 放松音乐、呼吸引导、减压小工具
- 📱 响应式设计，适配移动端与桌面端

## 技术栈
- 前端：HTML + CSS + JavaScript / Vue 
- AI：AI 大模型接口实现智能对话与情绪分析
- 构建工具：Vite 
- 部署：GitHub Pages / Netlify / Vercel

## 目标
用简单友好的界面 + AI 能力，
让每个人都能随时获得温暖、专业、私密的心理陪伴。
